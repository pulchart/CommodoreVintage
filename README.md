# CommodoreVintage

My useful scripts/tools/notes for commodore vintage C64/C128/Amiga1200 computers and C1571/C1581 drives.

* [Quieting the Commodore C1581 Disk Drive](https://github.com/pulchart/CommodoreVintage/tree/master/c1581-drive-quieting) with ROM signature/checksum fix.
* [fat95 - FAT filesystem handler for AmigaOS improvements](https://github.com/pulchart/fat95) fork of Torsten Jager's fat95 v3.18,
* [cfd -compactflash.device driver for CompactFlash cards in PCMCIA slots improvements](https://github.com/pulchart/cfd) fork of Torsten Jager's cfd133,
